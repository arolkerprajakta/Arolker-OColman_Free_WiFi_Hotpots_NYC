========================================================================================
INFM600-Info-Organiztion-Group-5 (Prajakta Arolker and Ryan O'Colman)
========================================================================================

This repository has been created as a part of an assignment for INFM600 course at 
University of Maryland, College Park.


----------------------------------------------------------------------------------------
• Original Dataset Source:

This dataset contains the reported Wi-Fi Hotspots throughout NYC
URL : https://data.cityofnewyork.us/Social-Services/NYC-Wi-Fi-Hotspot-Locations/a9we-mtpn

----------------------------------------------------------------------------------------
• Subset of the dataset: 

The dataset contains provider, location and names of free WiFi Hotspots reported in NYC.
URL : https://github.com/praj24/Arolker-OColman_Free_WiFi_Hotpots_NYC/
  
  • Data Statistics:
    - Size : 57.2 KB
    - Number of records : 610

----------------------------------------------------------------------------------------
• Research Question: 

Does the current free hotspot coverage offer everyone in New York City free coverage 
within 1 mile of their home?
